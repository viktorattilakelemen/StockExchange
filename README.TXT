PROJECT TITLE: StockExchange
PURPOSE OF PROJECT: measure runtime difference between ArrayList and BST
                    show proficiency using BSTs
VERSION or DATE:5/7/2018
HOW TO START THIS PROJECT: run StockDriver, then StockBSTDriver
AUTHORS: Viktor Kelemen
USER INSTRUCTIONS: Both drivers output a csv file, with the new prices of
each stock.
Delete previous output file before running driver again!!!
------------------------------FILES-------------------------------------
Stock - class that holds stock objects
StockDriver - reads in stocks from file, changes the stock prices based
on change.txt, using an ArrayList implementation
StockBSTDriver - reads in stock from file, changes the stock prices based
on change.txt using a TreeMap (BST) implementation
